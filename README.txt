ASHEN & MALSHIKA PREMIUM WEDDING WEBSITE

HOW TO USE:
1. Upload all files/folders to your GitHub repository root.
2. Make sure index.html is directly in the root folder.
3. Enable GitHub Pages: Settings > Pages > Deploy from branch > main > /(root).

HOW TO EDIT DETAILS:
- Open config.js.
- Edit names, date, time, venue, location, RSVP number, schedule and invitation message.

HOW TO CHANGE PHOTOS:
- Replace images in assets folder using the same filenames:
  groom.jpg
  bride.jpg
  groom_avatar.png
  bride_avatar.png
  hotel.jpg
  cartoon_couple.webp
  download.mp4

IMPORTANT:
- Video file must be named download.mp4.
- RSVP number is in config.js as 94769788163.
